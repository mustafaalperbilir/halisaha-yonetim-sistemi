# halisaha-yonetim-sistemi
 halı saha organizasyonlarını profesyonelce yönetmeyi sağlayan web tabanlı bir SaaS projesidir. Oyuncu verilerine dayalı akıllı takım dengeleme algoritması, maç tahmini, Telegram Bot entegrasyonu ve Firebase tabanlı güvenli veri yönetimi içerir. (Multi-tenant) yapısı sayesinde her kullanıcı kendi organizasyonunu ve bildirim ayarlarını yönetebilir.
